# regenrek.test-skill

Standalone reference skill published directly to the AgentRig registry. It
sources the same upstream content as the bundled `skill:test-submission`
inside `regenrek.test-submission`, but is published as its own signed artifact
to verify standalone-skill ingestion.

Source: https://github.com/regenrek/agentrig-test-plugin (v0.2.0,
`skills/test-submission/`).
