# regenrek.test-mcp

Standalone reference MCP server published directly to the AgentRig registry.
Mirrors the bundled `.mcp.json` from `regenrek.test-submission`, packaged as
its own signed artifact to verify standalone-MCP ingestion.

Source: https://github.com/regenrek/agentrig-test-plugin (v0.2.0,
`.mcp.json`).
