# Rust skill

Rust quality checks: fmt, clippy, tests.

## Installation

```bash
agentrig plugin install codex rust
```

## What's included

- `skills/rust/SKILL.md`
- `skills/rust/rust-check.sh`

## Provider install

- Install with any supported provider: `codex`, `claude`, or `cursor`.

## Usage

Run the Rust check workflow and apply recommendations in your repo.
