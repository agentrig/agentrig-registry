# Committer skill

Adds a PR committer skill with a safe commit helper script.

## Canonical Install Ref

`agentrig/agentrig.core-committer@0.1.0`

## What's included

- `skills/pr-commiter/SKILL.md`
- `skills/pr-commiter/scripts/pr-commiter.sh`

## Provider install

- Install with any supported provider: `codex`, `claude`, or `cursor`.

## Usage

Once installed, the committer skill is available to your agents and tooling.
