# Security check skill

Adds a security check skill that runs lightweight checks before shipping.

## Canonical Install Ref

`agentrig/agentrig.security-check@0.1.0`

## What's included

- `skills/security-check/SKILL.md`
- `skills/security-check/security-check.sh`

## Provider install

- Install with any supported provider: `codex`, `claude`, or `cursor`.

## Usage

Run the security checks before release and address any findings.
